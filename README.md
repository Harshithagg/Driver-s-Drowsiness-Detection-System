# Real-Time Driver Drowsiness Detection System

## Problem Statement
Driver fatigue is one of the leading causes of road accidents. Detecting drowsiness in real-time can help prevent accidents and improve road safety.

## Solution Overview
This project uses computer vision techniques to monitor a driver’s eye movements and detect signs of drowsiness. When drowsiness is detected, an alert is triggered.

## Features
- Real-time face and eye detection
- Drowsiness detection system
- Alert system (alarm)
- Live webcam monitoring

## Tech Stack
- Python
- OpenCV
- TensorFlow / Keras
- NumPy

## Installation
1. Clone the repository:
   git clone https://github.com/your-username/Driver-s-Drowsiness-Detection-System
2. Install dependencies:
   pip install -r requirements.txt
3. Run the project:
   python Driver's_drowsiness_detection_system.py

## Usage
- Run the program
- Allow webcam access
- The system detects eye closure
- Alert is triggered if drowsy
